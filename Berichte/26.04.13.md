## Übung (netcat chat)
Die Aufgabe ist eine bidirektionalen Chat zwischen zwei Rechnern aufzubauen. Dazu muss man netcat verwenden.

|IP-Adresse|Computer|
|----------|--------|
|192.168.13.51|PC-A|
|192.168.13.52|PC-B|

**Befehl am PC-A**
```sh
nc -lvp 1234
```

**Befehl am PC-B**
```sh
$ nc 192.168.13.51 1234
```
Hier wurde die IP-Adresse des anderen Rechners eingegeben und auf den Port 1234 gehorcht. Nun kann man alle Textzeichen die in den Chat geschrieben werden sehen.

## Übung (netcat data send)
Nun soll man über den Chat eine Textdatei übertragen, welche der andere dann abspeichern kann. 
Dazu muss der Sender im vorhinein ein Textfile mit Inhalt erstellt haben.

**Befehl am PC-A**
```sh
nc -lvp 1234 > empfangen.txt
```

**Befehl am PC-B**
```sh
$ nc 192.168.13.51 1234 < text.txt
```
Nun kann man am PC-A das File öffnen
